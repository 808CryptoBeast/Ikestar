## Ikeverse Night Sky Observatory

Ikeverse Night Sky Observatory is an interactive Three.js sky experience inspired by Hawaiian wayfinding. It blends real-time sky navigation, cultural sky layers, star stories, formation views, Moon and planet panels, and a Hawaiian star compass into one immersive observatory.

## Features

- Interactive real-time night sky
- Hawaiian, Western, Kemet, Polynesian, Aboriginal, and Ikeverse sky layers
- Clickable stars, Moon, and planets
- Formation Focus mode
- Story / Moʻolelo mode
- Navigation tools and seasonal sky elements
- Custom ʻIwa bird cursor
- Hawaiian star compass overlay

## Tech Stack

- HTML
- CSS
- JavaScript
- Three.js
- Font Awesome
